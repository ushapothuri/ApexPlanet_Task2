Task Manager Web Application

Overview

Task Manager is a simple and responsive web application built using HTML, CSS, and JavaScript. It allows users to manage daily tasks through a dynamic To-Do List and includes a Contact Form with client-side validation.

This project was developed to demonstrate fundamental web development concepts such as responsive design, form validation, DOM manipulation, and event handling.

---

## Features

### Contact Form

* Name input field
* Email input field
* Form validation using JavaScript
* Success and error messages

### To-Do List

* Add new tasks
* Delete existing tasks
* Dynamic task management using JavaScript

### Responsive Design

* Flexbox-based layout
* Mobile-friendly interface
* Media queries for smaller screens
